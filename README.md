# iAngels VC Platform - Frontend Project (React)

This repository contains the comprehensive project book and documentation for my internship project developed at **iAngels VC Platform** (Tel Aviv). 

## 🚀 Project Overview
During my internship, I was part of the Frontend Team, where I contributed to a regulated user-management system and financial forms for a venture capital platform managing **$400M+** across multiple investment products, supporting **20,000+ verified investors**.

## 🛠️ Tech Stack
* **Frontend:** React.js, JavaScript (ES6+), HTML5, CSS3
* **Testing:** Jest / React Testing Library
* **UI/UX:** Responsive Design (Mobile-First), Secure Document Uploads, Form Management

## ✨ Key Responsibilities & Implementations
* **Regulated User Management:** Enhanced financial forms and investor profile systems while ensuring strict regulatory compliance.
* **Complex Financial Workflows:** Developed and optimized multi-step KYC (Know Your Customer) forms and investment rollover mechanisms.
* **Security & Responsive Design:** Built secure, mobile-responsive interfaces for investor profiles, document upload flows, and identity verification.
* **Testing & Quality Assurance:** Developed comprehensive unit tests to maintain high test coverage for critical financial flows.
* **Production Contribution:** Collaborated closely with the core development team in an Agile environment to deliver production-level code.
